## Project Instructions

- All Huawei Health APK decompile, modification, rebuild, and related assets must be handled inside this repository: `git@github.com:Perdonus/huawei.git`.
- Do not perform the main reverse-engineering or rebuild workflow outside this repo. Temporary downloads may be staged elsewhere only if they are moved here before analysis/build steps continue.
- Do not run local APK build, rebuild, or verification loops for this project. Use GitHub Actions as the only build/verification path and inspect GitHub logs/artifacts for diagnostics.
